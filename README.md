# ElezioniVeneto2025
Dati e codici utilizzati per testare Claude Opus 4.5 rispetto alle elezioni in Veneto

Ho testato Claude Opus 4.5 per estrarre e visualizzare dati relativi alle elezioni regionali venete del 23 e 24 novembre 2025
-Il file Bepi.py è lo script usato per estrarre, sezione per sezione, i voti ottenuti dalla Lega e le preferenze ottenute dal governatore uscente Luca Zaia
-Il file risultati_veneto_2025 è il file contenente i dati estratti
-Gli altri file sono quelli utilizzati per realizzare una mappa coropletica che visualizza la percentuale di voti della Lega con la preferenza per Zaia

Il pezzo in cui racconto come ho fatto è uscito su InfoData24, il datablog del Sole24Ore, a questo link: https://www.infodata.ilsole24ore.com/abbiamo-testato-claude-opus-4-5-per-pesare-il-consenso-di-luca-zaia-in-veneto
